Створення десктопної версії:
Спочатку було створено базову HTML-структуру з семантичними тегами (header, main, footer і т.д.) та CSS-стилі. Для компонування використовувалися Flexbox.

Створення планшетної та мобільної версій:
Адаптація під менші екрани реалізована за допомогою CSS медіа-запитів (@media) з точками перелому на 1200px, 992px , 600px та 400рх.
Всередині медіа-запитів: Flexbox (display: flex, flex-wrap: wrap) застосовано для автоматичного перебудовування сітки карток.
Реалізовано бургер-меню (додано HTML-кнопку, CSS для вигляду та анімації виїзду, JavaScript для перемикання класів).
Перебудовано макети деяких секцій (напр., hero) для уникнення накладання елементів на вузьких екранах.
Скориговано відступи, розміри шрифтів та ширини блоків.
Використано відносні одиниці (vh, vw у clamp(), %, rem) та метатег viewport.
Додано CSS transition для плавних ефектів.

Логічні оператори: використовувався логічний оператор and у медіа-запитах (напр., @media (max-width: 992px) означає @media all and (max-width: 992px)).
