Завдання №1.

Збережіть введене значення часу (год:хвил) (використати метод prompt()) у змінну hour/ Вивести це значення у Console, використовуючи шаблонний рядок. Перевірити до якої чверті години відноситься введене значення. Відповідь вивести за допомогою alert(). Підказка: не забудьте зробити ряд перевірок.

Код:

                            
                            // 0-14 хв = 1-ша чверть
                            // 15-29 хв = 2-га чверть
                            // 30-44 хв = 3-тя чверть
                            // 45-59 хв = 4-та чверть
                            
                            let timeInput = prompt("Введіть час у форматі 'год:хвил' (наприклад, 10:30):");
                            
                            if (timeInput && timeInput.includes(':')) {
                                // Виведення у консоль
                                console.log(`Введений час: ${timeInput}`);
                            
                                // Отримуємо хвилини
                                let parts = timeInput.split(':');
                                let minutes = parseInt(parts[1], 10);
                            
                                if (!isNaN(minutes) && minutes >= 0 && minutes <= 59) {
                                    if (minutes >= 0 && minutes <= 14) {
                                        alert("Перша чверть години");
                                    } else if (minutes >= 15 && minutes <= 29) {
                                        alert("Друга чверть години");
                                    } else if (minutes >= 30 && minutes <= 44) {
                                        alert("Третя чверть години");
                                    } else { // 45-59
                                        alert("Четверта чверть години");
                                    }
                                } else {
                                    alert("Некоректно введено хвилини.");
                                }
                            } else {
                                console.log("Введення скасовано або має неправильний формат.");
                                alert("Неправильний формат введення. Потрібен формат 'год:хвил'.");
                            }
