Завдання №3.

Є декілька користувачів User1, User2, User3, з відповідними паролями. (Об’явити , проініцілізувати) Написати програмний код, який запитуватиме у користувача логін (використати prompt) і виводить результат в консоль браузера, використовуючи шаблонний рядок. Після цього ввести логін користувача. Додайте перевірку введеного користувачем значення:  якщо відвідувач вводить, наприклад, " User1" або " User2" або" User3", то prompt запитує пароль. У випадку вдалої ідентифікації в alert вивести « Hello, User1» або « Hello, User2»; або « Hello, User3»;;

Код:

                            
                            const users = {
                                'User1': 'pass1',
                                'User2': 'pass2',
                                'User3': 'pass3'
                            };
                            
                            let login;
                            let isLoggedIn = false;
                            
                            // Цикл буде повторюватись, доки логін не буде успішним,
                            // або користувач не введе щось, що призведе до "I don't know you"
                            while (!isLoggedIn) {
                                login = prompt("Введіть ваш логін (User1, User2, User3):");
                            
                                if (login === null || login === '') {
                                    // Якщо натиснули Cancel або нічого не ввели,
                                    // prompt з'явиться знову (згідно з умовою)
                                    console.log("Спроба входу скасована, повторний запит...");
                                    continue; // Повертаємось на початок циклу
                                }
                            
                                // Виведення логіну в консоль
                                console.log(`Користувач спробував увійти з логіном: ${login}`);
                            
                                // Перевірка, чи існує такий користувач
                                if (login === 'User1' || login === 'User2' || login === 'User3') {
                                    let password = prompt(`Введіть пароль для ${login}:`);
                                    
                                    // Перевіряємо пароль з об'єкта users
                                    if (password === users[login]) {
                                        alert(`Hello, ${login}`);
                                        isLoggedIn = true; // Виходимо з циклу
                                    } else {
                                        alert("Невірний пароль!");
                                        // Залишаємось у циклі, щоб дати спробувати ще раз
                                    }
                                } else {
                                    alert("I don't know you");
                                    break; // Виходимо з циклу, якщо користувача не знайдено
                                }
                            }
