1.1. Адаптивна сітка контенту (Flexbox):

У моїй роботі я застосувала гумовий дизайн веб-сайту, що забезпечує його адаптивність до різних розмірів екранів. Для компонування контенту використовувалася технологія CSS Flexbox, яка дозволяє гнучко вирівнювати елементи у контейнерах, автоматично адаптуючи їх до змін умов відображення. Це забезпечує ефективне розміщення контенту на сторінці, зручне для перегляду з будь-якого пристрою.

1.2. CSS-код :

.houses-grid {
    display: flex; /* Вмикаємо Flexbox */
    flex-wrap: wrap; /* Дозволяємо перенос */
    gap: 40px;
}
.house-card {
flex: 1 1 30%; /* Гнучкість: рости, стискатись, базова ширина */
min-width: 300px; /* Мінімальна ширина */

}




2.1. Адаптивна навігація (бургер-меню):

Було реалізовано "бургер-меню": додано відповідну кнопку в HTML, стилізовано її показ/приховування та виїзд повноекранного меню за допомогою CSS, а також написано невеликий JavaScript-код для обробки кліків та перемикання класів, що керують видимістю меню.

2.2. HTML-код :

        <nav aria-label="Головна навігація">
        <button class="burger-menu" aria-label="Відкрити/закрити меню" aria-expanded="false">
        <span></span>
        <span></span>
        <span></span>
        </button>
        <ul class="nav-links">
        ...
        </ul>
        </nav>

2.3. CSS-код :

                                    /* Базові стилі */
                                    .burger-menu {
                                    display: none; /* Ховаємо на десктопі */
                                    }
                                    
                                    /* Медіа-запит для планшетів і менше */
                                    @media (max-width: 992px) {
                                    .burger-menu {
                                    display: block; /* Показуємо бургер */
                                    }
                                    .header-contacts {
                                    display: none; /* Ховаємо контакти */
                                    }
                                    nav ul.nav-links {
                                    position: fixed; /* На весь екран */
                                    /* ... інші стилі позиціонування ... */
                                    transform: translateX(100%); /* Ховаємо за екран */
                                    transition: transform 0.4s ease-in-out;
                                    }
                                    nav ul.nav-links.active {
                                    transform: translateX(0); /* Показуємо (виїжджає) */
                                    }
                                    }

2.4. JS-код :

                                // main.js
                                const burgerMenu = document.querySelector('.burger-menu');
                                const navLinks = document.querySelector('.nav-links');
                                
                                burgerMenu.addEventListener('click', () => {
                                navLinks.classList.toggle('active'); // Перемикаємо клас для меню
                                burgerMenu.classList.toggle('active'); // Перемикаємо клас для анімації кнопки
                                });




3.1. Відносні одиниці та анімація:

Також було застосовано відносні одиниці вимірювання (такі як vh, vw, %, rem) для деяких елементів та шрифтів, що сприяє гнучкості макету, а метатег viewport забезпечує правильне масштабування на мобільних пристроях. Для покращення користувацького досвіду було додано CSS-переходи (transition), які забезпечують плавність візуальних ефектів при наведенні курсора та анімацію бургер-меню.

3.2. HTML :

                                <meta name="viewport" content="width=device-width, initial-scale=1.0">

3.2. CSS :

                                #hero { height: 100vh; } /* vh */
                                @media (max-width: 1200px) { #hero { min-height: 70vh; } }
                                
                                .section-title .line1,
                                .section-title .line2 {
                                font-size: clamp(70px, 8vw, 112px); /* vw */
                                }
                                
                                .about-table { max-width: 60%; } /* % */
                                .section-title .line2-wrap { margin-left: 15%; } /* % */
                                
                                .burger-menu { padding: 0.5rem; margin-left: 1rem; } /* rem */

3.2. CSS (Анімація):

                                    a { transition: 0.3s ease; }
                                    .btn-book { transition: 0.25s; }
                                    .house-card { transition: transform 0.3s ease, box-shadow 0.3s ease; }
                                    .burger-menu span { transition: transform 0.3s ease, background-color 0.3s ease; }
                                    nav ul.nav-links { transition: transform 0.4s ease-in-out; }
                                    
                                    a:hover { opacity: 0.7; }
                                    .btn-book:hover { background: #3C3126; color: #fff; }
                                    .house-card:hover { transform: translateY(-5px); }                                                                                          });




4.1. Медіа-запити (media queries):

Для реалізації специфічних змін макету та стилів на різних пристроях було використано медіа-запити (@media queries). Всередині них коригувалися відступи, розміри шрифтів, а також змінювався макет деяких секцій (наприклад, hero секції, щоб уникнути накладання елементів) та реалізовувалася адаптивна навігація.

4.2. CSS :

                                        /* === 0. ПРОМІЖНИЙ РОЗМІР === */
                                        @media (max-width: 1200px) {
                                        #hero { display: flex; /* і т.д. */ }
                                        .section-title { /* Коригування */ }
                                        }
                                        
                                        /* === 1. ПЛАНШЕТ (Портрет і менше) === */
                                        @media (max-width: 992px) {
                                        .burger-menu { display: block; }
                                        nav ul.nav-links { position: fixed; /* і т.д. */ }
                                        }
                                        
                                        /* === 2. ТЕЛЕФОН (і менше) === */
                                        @media (max-width: 600px) {
                                        .hero-content h1 { font-size: 40px; }
                                        }
